

## 콘텐츠 확장 계획

4가지 새로운 콘텐츠를 별도 페이지로 추가합니다. 메인 페이지에서 네비게이션으로 접근할 수 있도록 합니다.

### 1. 새 페이지 4개 생성

| 페이지 | 경로 | 설명 |
|--------|------|------|
| **한글 포스터 갤러리** | `/posters` | CSS로 디자인된 시각적 포스터 카드 (한글 자음/모음을 활용한 타이포그래피 아트 6-8장) |
| **한글 역사 타임라인** | `/history` | 1443년 창제부터 현대까지 주요 사건을 세로 타임라인으로 표현 |
| **한글날 소개** | `/hangulday` | 한글날의 의미, 유네스코 등재, 세계 속 한글 위상 소개 |
| **명언/문구 컬렉션** | `/quotes` | 한글 관련 아름다운 명언과 시를 카드 형태로 전시 |

### 2. 네비게이션 업데이트

- `Navbar.tsx`에 "더 알아보기" 드롭다운 메뉴 추가 (포스터, 역사, 한글날, 명언 링크)
- react-router-dom `Link` 사용하여 페이지 이동

### 3. 라우팅 업데이트

- `App.tsx`에 4개 새 라우트 추가

### 4. 공통 레이아웃

- 각 페이지에 Navbar + FooterSection 포함
- 메인 페이지와 동일한 한지 텍스처 + 붓글씨 스타일 유지
- Framer Motion으로 스크롤 애니메이션 적용

### 5. 콘텐츠 상세

- **포스터 갤러리**: CSS-only 타이포그래피 포스터 (한글 자모를 대형 폰트로 배치, 배경색/레이아웃 변형)
- **타임라인**: 세종대왕(1443) → 반포(1446) → 주시경(1910) → 한글날 제정(1926) → 유네스코 등재(1997) 등 8-10개 이벤트
- **한글날**: 10월 9일의 역사적 의미, 세계문자올림픽, 찌아찌아족 한글 채택 등
- **명언**: 세종대왕 어제서문, 주시경 선생 명언, 현대 문인들의 한글 관련 문구 등 8-10개
